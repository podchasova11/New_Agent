

## New_Agent

**Dulus** — это open-source автономный оркестратор AI-агентов (CLI-инструмент на Python).

Суть: вы устанавливаете его (`pip install dulus`), подключаете любой LLM (Claude, GPT, Gemini, DeepSeek, Ollama и др.) и получаете мощную агентскую среду в терминале. Агент может:

- читать/писать файлы, выполнять bash-команды, делать веб-поиск
- запускать **суб-агентов** (coder, reviewer, tester) в параллельных git-ветках
- работать с **MCP-серверами** и **плагинами** (авто-адаптация любого Python-репо)
- хранить **память** между сессиями, откатываться через **чекпоинты**
- принимать **голосовой ввод** (Whisper, офлайн) и управляться через **Telegram**
- работать **без API-ключа** — захватывая гостевую сессию Gemini/Claude.ai через браузер

Это форк репозитория KevRojo/Dulus — сам проект создан одним разработчиком (@KevRojo) и позиционируется как более дешёвая альтернатива Claude Code с мультипровайдерной поддержкой.

Вот план — бесплатно, на Windows, без API-ключа.

---

## Шаг 1 — Установка Python (если нет 3.11+)

Проверь версию в командной строке:
```
python --version
```
Если меньше 3.11 — скачай с [python.org](https://python.org) и при установке **обязательно поставь галочку "Add to PATH"**.

---

## Шаг 2 — Установка Dulus

Открой **PowerShell** и запусти:
```powershell
pip install dulus
```
Или одной командой (авто-установщик):
```powershell
irm https://raw.githubusercontent.com/KevRojo/Dulus/main/install.ps1 | iex
```
Выбери профиль **`basic`** — он самый лёгкий (~150 МБ).

---

## Шаг 3 — Запуск без API-ключа (бесплатно через Gemini)

```powershell
dulus
```
При первом запуске мастер настройки спросит про ключ. Выбери вариант **"открыть Gemini в браузере"** — Dulus захватит гостевую сессию автоматически. Это не требует ни аккаунта Google, ни карты.

Альтернатива — полностью офлайн через **Ollama**:
```powershell
# 1. Скачай Ollama с ollama.com, установи
# 2. Скачай модель
ollama pull qwen2.5-coder

# 3. Запусти Dulus с этой моделью
dulus --model ollama/qwen2.5-coder
```

---

## Шаг 4 — Первые команды в REPL

Когда Dulus запустился и ты видишь приглашение `▲` — можно работать:

| Что написать | Что произойдёт |
|---|---|
| `Привет, что ты умеешь?` | просто чат с агентом |
| `/help` | список всех команд |
| `/model` | показать текущую модель |
| `/cost` | сколько токенов потрачено |
| `/memory search код` | поиск по памяти |
| `/checkpoint` | сохранить состояние |

---

## Шаг 5 — Попробуй реальную задачу

Напиши агенту что-то вроде:
```
Создай файл hello.py с функцией, которая считает факториал, и напиши тест к ней
```
Dulus сам создаст файл, напишет код и тест. Ты можешь одобрять или отклонять каждое действие.

---

## Важно про режимы разрешений

По умолчанию Dulus **спрашивает перед каждым действием** с файлами. Если хочешь, чтобы делал всё сам — добавь флаг:
```
dulus --accept-all
```
Но на старте лучше без него — так видно, что агент делает.
